# Proyecto Final

> **Curso:** Introducción a la Ciencia de Datos  
> **Dataset:** Telco Customer Churn — Kaggle

## Fases del proyecto

| # | Fase CRISP-DM | Descripción |
|---|---|---|
| 1 | **Comprensión del Negocio** | Definición del objetivo: predecir el abandono de clientes (*churn*). Identificación de stakeholders y criterios de éxito del proyecto. |
| 2 | **Comprensión de los Datos** | Análisis exploratorio del dataset Telco Customer Churn: estadísticas descriptivas, calidad de datos y visualizaciones de variables clave. |
| 3 | **Preparación de los Datos** | Limpieza de valores nulos, codificación de variables categóricas, escalado de variables numéricas y balanceo de clases. |
| 4 | **Modelado** | Entrenamiento y ajuste de mínimo dos modelos de clasificación supervisada para predecir la variable *Churn*. |
| 5 | **Evaluación** | Comparación de modelos mediante métricas como AUC-ROC, F1-score y Recall. Selección del modelo con mejor desempeño. |
